# ☁️ VPS авч, Kodu Sandbox-ыг 24/7 онлайн болгох заавар

> Энэ бол sandbox замын сүүлийн том алхам: engine-ийг гэрийн компьютерээс
> **жинхэнэ сервер** рүү нүүлгэнэ. Дараа нь домэйн + KoDu холболт нээгдэнэ.

## 1. VPS гэж юу вэ

VPS = интернэтэд 24/7 залгаастай, түрээсийн жижиг компьютер. Бид дээр нь
Ubuntu Linux + Docker + controller-оо суулгана. **Сард ~₮15–35 мянга.**

## 2. Аль компанийг сонгох вэ

| Компани | Байршил | Үнэ (сар) | Тэмдэглэл |
|---------|---------|-----------|-----------|
| **Hetzner** ⭐ үнэ/чанар | Герман, Финлянд, Сингапур | €4.5~ (2 vCPU / 4GB) | Хамгийн хямд, чанартай. Бүртгэлдээ баримт шаардаж магадгүй |
| **Vultr** ⭐ Азид ойр | Токио, Сөүл, Сингапур | $6–12 | Alipay/карт хоёуланг авдаг, Азиас хурдан |

**Зөвлөмж:** Монголоос хэрэглэгчидтэй бол **Токио/Сөүл/Сингапур** байршил
сонго (хоцролт бага). Хамгийн хямдыг хүсвэл Hetzner.

**Хэмжээ:** Next.js preview контейнер бүр 1GB RAM авдаг тул **дор хаяж
2GB RAM, боломжтой бол 4GB** сонго. OS нь **Ubuntu 24.04** байг.

## 3. Бүртгүүлээд сервер үүсгэх (ерөнхий алхам)

1. Сонгосон компанийн сайтад бүртгүүл (имэйл + карт эсвэл Alipay)
2. "Deploy / Create server" дар
3. Сонго: **Ubuntu 24.04** · байршил (Токио г.м) · хэмжээ (2–4GB RAM)
4. Нэвтрэлт: **Password** сонговол хялбар (SSH key нь илүү аюулгүй —
   дараа шилжиж болно)
5. Үүсгэмэгц танд **IP хаяг** (жишээ: `203.0.113.45`) + **root нууц үг** өгнө

## 4. Суулгах — ГАНЦ КОМАНД

Windows PowerShell дээрээс сервер лүү холбогдо:

```powershell
ssh root@<СЕРВЕРИЙН-IP>
```

(эхний удаад "yes" гэж зөвшөөрч, нууц үгээ оруулна)

Тэгээд сервер дотор энэ **ганц командыг** ажиллуул:

```bash
curl -fsSL https://raw.githubusercontent.com/Zulzaga0428/SAND/main/deploy/setup.sh | bash
```

Скрипт өөрөө: Docker + Node суулгаж, repo татаж, template build хийж,
controller-ыг **systemd service** болгоно (сервер unтраад асахад өөрөө асна).

Дуусахад хэвлэнэ:
```
🐳 Kodu Sandbox VPS дээр АМЬД боллоо!
Самбар:      http://<IP>:4000
API түлхүүр: kodu_xxxxx...
```

## 5. Шалгах

- Browser дээр `http://<IP>:4000` нээ → самбар гарна (түлхүүр асууна)
- Preview үүсгэж үз → `http://<IP>:<порт>` хэлбэрийн URL ирнэ
- Лог харах: `journalctl -u kodu-sandbox -f`
- Дахин асаах: `systemctl restart kodu-sandbox`

## 6. Дараагийн алхмууд (VPS амьд болсны дараа)

- [ ] Домэйн авах (жишээ: kodusandbox.mn эсвэл хямд .com/.dev)
- [ ] DNS: `api.домэйн` → VPS IP, `*.p.домэйн` → VPS IP (wildcard)
- [ ] Reverse proxy (Caddy) — HTTPS + цэвэр preview URL (M3)
- [ ] Гадагшаа сүлжээг iptables-ээр хаах (аюулгүй байдлын сүүлийн алхам)
- [ ] KoDu-д client бичих → E2B-тэй зэрэгцээ shadow-тест (M5)
