Боломжууд

Түр зуурын контейнер

Preview бүр өөрийн тусгаарлагдсан контейнерт. Ажил дуусмагц ор мөргүй устана — үлдэгдэл, хог байхгүй.

⚛️

Next.js template

node_modules урьдчилан суулгасан image — npm install огт хийхгүй, React апп секундэд асна.

🔑

API түлхүүр

Бүх хүсэлт Bearer түлхүүрээр хамгаалагдсан. Оролтын хатуу хязгаар: файлын тоо, хэмжээ шалгагдана.

🔒

Нөөцийн хязгаар

Контейнер бүр RAM/CPU/process хязгаартай, root биш хэрэглэгчээр ажиллана. Нэг нь нөгөөд халдахгүй.

📁

Олон файлтай төсөл

HTML/CSS/JS-ээс эхлээд бүтэн Next.js төсөл хүртэл — файлуудыг tar-аар контейнерт шууд бичнэ.

🇲🇳

Монголд, монголоор

Монголын AI вэб бүтээгч KoDu-гийн preview-г тэжээх зорилготой төрсөн. Баримт бичиг бүгд монголоор.

Хэрхэн ажилладаг вэ

Гурван алхам — API дуудахад бүгд автоматаар болно.

1

Файлуудаа илгээ. POST /api/previews руу төслийн файлууд + горимоо (static / Next.js) явуулна.

2

Sandbox контейнер асаана. Файлууд tar-аар контейнерт бичигдэж, сервер ажиллаж эхэлмэгц бэлэн болно.

3

Preview URL авна. Хариунд шууд нээж болох URL ирнэ. 15 минутын дараа контейнер өөрөө цэвэрлэгдэнэ.
